# PaperEveryday 📄

> 个人论文追踪 Dashboard — 快速跟进研究领域的最新进展。

纯前端静态网页，通过 JSON 文件管理论文，配合 GitHub Actions 自动拉取 ArXiv 最新论文，可部署到 GitHub Pages。

在线访问：**[https://zhousf-628.github.io/PaperEveryday/](https://zhousf-628.github.io/PaperEveryday/)**
